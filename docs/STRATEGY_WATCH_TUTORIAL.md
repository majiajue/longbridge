# 策略盯盘使用教程 🎯

## 快速开始（3分钟上手）

### 第1步：访问策略盯盘页面

1. 打开浏览器访问：`http://localhost:5173`
2. 点击顶部导航栏的 **「策略盯盘 🎯」** Tab

**📌 重要说明：**
- 系统会**自动监控**你的持仓股票 + 在"基础配置"中添加的股票
- 持仓股票会优先显示，并带有 **"持仓"** 标记
- 如果没有持仓也没有添加股票，页面会提示你添加

---

### 第2步：查看信号概览

进入后你会看到两个标签页：

#### 📊 信号概览（推荐先看这个）

**这是什么？**
- 显示所有监控股票的实时交易信号
- 一眼看清哪些股票该买、该卖、该观望

**界面说明：**

```
┌──────────────────────────────────────────────────────────────────┐
│  策略盯盘                      [持仓 2 | 监控 1] [共 3 只] [刷新] │
│  实时监控买低卖高和 EMA 策略信号                                  │
└──────────────────────────────────────────────────────────────────┘

┌─ 信号概览 ─┬─ 实时 K 线 ─┐
│                                                                  │
│  股票           当前价   买低卖高   EMA交叉   综合建议   置信度   │
│  ──────────────────────────────────────────────────────────────  │
│  AAPL.US [持仓]  $175.50  🟢BUY    🟢BUY    🟢 BUY    ███ 72%   │
│  TSLA.US [持仓]  $245.80  🔴SELL    -       🔴 SELL   ██  55%   │
│  NVDA.US        $495.20    -       🟢BUY    🟢 BUY    ██  60%   │
└──────────────────────────────────────────────────────────────────┘
```

**表格各列含义：**

1. **股票**：股票代码（带 **[持仓]** 标记表示实际持有该股票）
2. **当前价**：最新价格
3. **买低卖高**：买低卖高策略的信号
   - 🟢 BUY = 检测到波谷，建议买入
   - 🔴 SELL = 检测到波峰，建议卖出
   - `-` = 暂无信号
4. **EMA交叉**：均线交叉策略的信号
   - 🟢 BUY = 金叉（快线上穿慢线）
   - 🔴 SELL = 死叉（快线下穿慢线）
   - `-` = 暂无信号
5. **综合建议**：两个策略的共识
   - 🟢 BUY = 多数策略建议买入
   - 🔴 SELL = 多数策略建议卖出
   - ⚪ HOLD = 观望或策略冲突
6. **置信度**：信号可靠程度（柱状图）
   - 绿色长条 = 高置信度（>70%）
   - 黄色中条 = 中等（50-70%）
   - 红色短条 = 低置信度（<50%）

## 进阶使用

### 如何添加监控股票？

策略盯盘会自动分析你在「基础配置」中添加的股票。

**步骤：**
1. 切换到 **「基础配置」** Tab
2. 在「股票代码列表」输入框添加股票
   ```
   AAPL.US
   TSLA.US
   NVDA.US
   700.HK
   ```
3. 点击 **「保存」**
4. 回到 **「策略盯盘」**，等待几秒自动刷新

---

### 如何理解信号？

#### 1️⃣ 买低卖高信号

**原理**：在价格波动中找到局部最低点（买入）和最高点（卖出）

**示例：**
```
✅ BUY 信号
检测到局部最低点 $173.20，当前反弹至 $175.50
置信度: 72%

说明：
- 股价在 $173.20 形成了一个"坑"（前后5天都比它高）
- 现在已经开始反弹
- 成交量确认有效
- 建议此时买入
```

```
🔴 SELL 信号
检测到局部最高点 $185.60，当前回落至 $183.20
置信度: 68%

说明：
- 股价在 $185.60 形成了一个"峰"（前后5天都比它低）
- 现在已经开始回落
- 成交量放大确认顶部
- 建议此时卖出
```

#### 2️⃣ EMA 交叉信号

**原理**：快线（短期均线）和慢线（长期均线）交叉判断趋势

**示例：**
```
✅ 金叉（BUY）
EMA12 上穿 EMA26
当前趋势: ↗️ 上升
置信度: 75%

说明：
- 短期均线突破长期均线，形成"金叉"
- 表示上升趋势开始
- MACD 指标同步转正
- 适合趋势跟踪买入
```

```
🔴 死叉（SELL）
EMA12 下穿 EMA26
当前趋势: ↘️ 下降
置信度: 70%

说明：
- 短期均线跌破长期均线，形成"死叉"
- 表示下降趋势开始
- MACD 指标同步转负
- 建议离场避险
```

---

### 如何使用综合建议？

系统会综合两个策略给出最终建议：

#### 🟢 强烈买入（置信度 > 70%）
```
两个策略都发出 BUY 信号
✓ 买低卖高：波谷反弹
✓ EMA 交叉：金叉确认
→ 建议：积极买入
```

#### 🟡 谨慎买入（置信度 50-70%）
```
一个策略 BUY，另一个无信号
✓ 买低卖高：BUY
- EMA 交叉：暂无信号
→ 建议：小仓位试探
```

#### 🔴 建议卖出
```
多数策略发出 SELL 信号
✓ 买低卖高：波峰回落
✓ EMA 交叉：死叉确认
→ 建议：及时止盈/止损
```

#### ⚪ 观望
```
策略冲突或都无信号
✓ 买低卖高：BUY
✗ EMA 交叉：SELL
→ 建议：等待更明确信号
```

## 常见问题 FAQ

### Q1: 为什么看不到信号？

**可能原因：**
1. ❌ 没有添加监控股票
   - 解决：去「基础配置」添加股票代码
   
2. ❌ 数据不足
   - 解决：等待系统同步历史数据（需要至少30根K线）
   
3. ❌ 当前无符合条件的信号
   - 解决：这是正常的，不是每只股票每天都有信号

### Q2: 信号会延迟吗？

**答：** 会有轻微延迟（约1-2根K线）

这是设计如此：
- 策略使用**倒数第2根K线**检测信号
- 避免使用未完成的K线导致虚假信号
- 确保信号更可靠

### Q3: 置信度怎么看？

**答：** 置信度越高越好

- 🟢 **70%+ = 强信号**：可以积极操作
- 🟡 **50-70% = 中等**：谨慎小仓位
- 🔴 **<50% = 弱信号**：建议观望

**置信度受这些因素影响：**
- 波幅大小（越大越好）
- 成交量确认（放量更可靠）
- 趋势一致性（趋势明确更可靠）

### Q4: 两个策略冲突怎么办？

**答：** 以置信度高的为准，或观望

**示例：**
```
买低卖高：BUY (60%)
EMA 交叉：SELL (75%)

建议：听 EMA 的，卖出或观望
```

**原则：**
1. 置信度差距大 → 听高的
2. 置信度接近 → 观望等待
3. 都是低置信度 → 不操作

### Q5: 可以自动交易吗？

**答：** 可以，但需要配置

**步骤：**
1. 去「持仓监控」页面
2. 为持仓股票设置策略
3. 选择「自动执行」模式
4. 系统会根据信号自动下单

**建议：**
- 新手先用「仅提醒」模式熟悉
- 确认策略有效后再开自动

### Q6: 多久刷新一次？

**答：** 自动刷新30秒

- 信号概览：每30秒刷新
- 实时K线：通过WebSocket实时推送（毫秒级）
- 手动刷新：点击右上角「刷新」按钮

## 使用技巧 💡

### 技巧1：多时间周期配合

```
日线图（策略盯盘）：判断大趋势
  └─ 如果显示上升趋势 ✓
  
小时线（自己分析）：寻找入场点
  └─ 等待回调买入
```

### 技巧2：结合持仓使用

```
1. 策略盯盘：发现买入信号
2. 持仓监控：配置止损止盈
3. 实时K线：跟踪价格变化
4. 策略盯盘：等待卖出信号
```

### 技巧3：批量监控

```
添加多只股票 → 策略盯盘自动分析
  ↓
只关注有信号的股票（表格会自动突出）
  ↓
点击「查看K线」深入分析
```

### 技巧4：信号确认

**不要盲目跟随单一信号！**

✅ 好的操作：
```
1. 策略盯盘显示买入信号
2. 查看实时K线确认趋势
3. 检查成交量是否配合
4. 确认置信度 > 60%
5. 执行买入
```

❌ 不好的操作：
```
1. 看到 BUY 就买
2. 不管置信度
3. 不看K线图
4. 结果：可能是假信号
```

## 注意事项 ⚠️

### ⚠️ 风险提示

1. **策略非万能**
   - 基于历史数据，不保证未来
   - 需要结合基本面分析
   - 市场环境变化策略可能失效

2. **严格止损**
   - 每个信号都有止损建议
   - 必须严格执行
   - 不要抱有侥幸心理

3. **控制仓位**
   - 单只股票不超过总资金的20%
   - 分散投资降低风险
   - 留有充足现金储备

4. **不要追高**
   - 错过就错过，不要追
   - 市场永远有机会
   - 耐心等待下一个信号
