## 🛠️ 工具推荐

### macOS
- **系统截图**：`Cmd + Shift + 4`（区域截图）
- **全屏截图**：`Cmd + Shift + 3`
- **窗口截图**：`Cmd + Shift + 4` 然后按 `Space`
- **推荐工具**：Snagit、CleanShot X

### Windows
- **系统截图**：`Win + Shift + S`（截图工具）
- **全屏截图**：`PrtScn`
- **推荐工具**：Snagit、ShareX

### Linux
- **系统截图**：`PrtScn` 或 `Shift + PrtScn`
- **推荐工具**：Flameshot、Shutter

## 🎨 后期处理

### 1. 图片编辑

**推荐工具**：
- **在线**：Photopea (https://www.photopea.com/)
- **桌面**：GIMP、Photoshop、Affinity Photo

**处理建议**：
- 裁剪不需要的部分（如浏览器工具栏）
- 调整亮度/对比度（如果需要）
- 添加边框或阴影（可选）
- 压缩文件大小（保持清晰度）

### 2. 图片压缩

**推荐工具**：
- **在线**：TinyPNG (https://tinypng.com/)
- **命令行**：ImageMagick

```bash
# 使用 ImageMagick 压缩
convert input.png -quality 85 -resize 1920x output.png
```

### 3. 图片尺寸建议

| 类型 | 宽度 | 高度 | 用途 |
|------|------|------|------|
| 全景 | 1920px | 1080px | 主界面、双栏布局 |
| 卡片 | 800px | 自适应 | 股票卡片详情 |
| 终端 | 1200px | 自适应 | 启动日志 |
