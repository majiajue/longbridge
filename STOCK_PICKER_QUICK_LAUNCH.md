# 🚀 智能选股快速启动指南

## 立即开始（3步完成）

### 第1步：启动系统

```bash
cd /Volumes/SamSung/longbridge
./start.sh
```

**期望输出**：
```
✅ 后端启动：http://localhost:8000
✅ 前端启动：http://localhost:5173
```

---

### 第2步：导入示例股票

**方式A：使用预置的40只股票**

已经内置在系统中！直接跳到第3步。

**方式B：自定义股票列表**

编辑 `import_stocks.py`，然后运行：

```bash
python import_stocks.py
```

**示例输出**：
```
📊 开始导入股票池...
✅ 做多池：成功 20，失败 0
✅ 做空池：成功 20，失败 0
```

---

### 第3步：打开前端使用

1. **打开浏览器**：
   ```
   http://localhost:5173
   ```

2. **点击「🎯 智能选股」标签**

3. **查看界面**：
   ```
   ┌───────────────────────────────────────┐
   │  📊 智能选股分析    [🔄 分析全部]     │
   ├──────────────────┬────────────────────┤
   │  📈 做多股票池    │   📉 做空股票池    │
   │  (20/20)         │   (20/20)          │
   │                  │                    │
   │  [➕ 添加]        │   [➕ 添加]        │
   │  [🔍 分析]       │   [🔍 分析]        │
   └──────────────────┴────────────────────┘
   ```

4. **触发分析**：
   - 点击顶部的「🔄 分析全部」按钮
   - 或单独点击某个池的「🔍 分析」按钮

5. **等待2-5分钟**（取决于股票数量）

6. **查看结果**：
   - 自动按推荐度排序
   - 查看评分、评级、推荐理由
   - 点击「▼ 查看详情」展开详细分析

## 💡 使用技巧

### 技巧1：优先关注高推荐度
- **推荐度 ≥ 80分**：重点关注
- **推荐度 70-79分**：可以考虑
- **推荐度 < 70分**：谨慎对待

### 技巧2：结合多个维度
不要只看评级，要看：
- ✓ 量化评分（技术面）
- ✓ AI信心度（综合判断）
- ✓ 推荐理由（逻辑）
- ✓ 技术信号（具体指标）

### 技巧3：定期重新分析
- 每天开盘前分析一次
- 盘中重要时点再分析
- 观察评分变化趋势

### 技巧4：灵活管理股票池
- 删除D级、C级股票
- 添加新的机会
- 保持池子活跃

## 🐛 故障排查

### 问题1：前端无法访问

**解决**：
```bash
# 检查端口是否被占用
lsof -i :5173

# 重启前端
cd frontend
npm run dev
```

### 问题2：后端API报错

**解决**：
```bash
# 查看后端日志
tail -f logs/backend.log

# 检查后端是否运行
curl http://localhost:8000/api/stock-picker/pools
```

### 问题3：分析失败

**原因可能**：
- DeepSeek API额度不足
- 股票代码格式错误
- 无K线数据

**解决**：
```bash
# 查看详细错误
tail -f logs/backend.log

# 检查API配置
curl http://localhost:8000/api/settings/credentials
```

### 问题4：股票池为空

**解决**：
```bash
# 运行导入脚本
python import_stocks.py

# 或手动添加
# 前端：点击「➕ 添加」
```
