## 📋 快速启动检查清单

### 1️⃣ 启动后端服务
```bash
cd /Volumes/SamSung/longbridge
# 确保后端正在运行
# 如果没有，运行：sh start.sh
```

### 2️⃣ 访问前端
```
http://localhost:5173
```

### 3️⃣ 配置凭据（Settings 页面）

**Longbridge 凭据：**
- [ ] APP_KEY
- [ ] APP_SECRET
- [ ] ACCESS_TOKEN
- [ ] 点击「验证凭据」✅

**AI 配置：**
- [ ] DeepSeek API Key
- [ ] 点击「测试 AI 连接」✅

### 4️⃣ 配置 AI 交易（AI Trading 页面）

点击右上角 ⚙️ **设置**：

```json
{
  "enabled": true,
  "symbols": ["AAPL", "TSLA", "MSFT"],  // 修改为你想监控的股票
  "check_interval_minutes": 5,
  "ai_model": "deepseek-chat",
  "ai_temperature": 0.3,
  "min_confidence": 0.70,              // 推荐起始值
  "max_daily_trades": 10,              // 限制交易次数
  "max_loss_per_day": 2000,            // 每日最大亏损
  "fixed_amount_per_trade": 5000,      // 单笔交易金额
  "enable_real_trading": false         // ⚠️ 先用模拟模式！
}
```

点击「保存配置」。

### 5️⃣ 清理旧数据（如有）

切换到「💼 持仓管理」标签：
- [ ] 点击「清空所有持仓」（清理之前的模拟数据）

### 6️⃣ 启动引擎

- [ ] 点击 **「▶️ 启动引擎」** 按钮
- [ ] 确认引擎状态变为「运行中 🟢」

### 7️⃣ 触发首次分析

- [ ] 点击 **「⚡ 立即分析」** 按钮
- [ ] 观察实时日志和 AI 决策

## 📊 模拟交易测试（1-2天）

### 观察指标：

1. **AI 决策频率**
   - 多久触发一次 BUY/SELL？
   - 大部分是 HOLD 吗？

2. **信心度分布**
   - 平均信心度是多少？
   - 是否有 ≥0.75 的高信心交易？

3. **K线形态识别**
   - AI 能否识别经典形态？
   - 形态识别准确吗？

4. **模拟盈亏**
   - 虽然是模拟，但观察盈亏趋势
   - 止损止盈是否合理？

### 调整参数：

根据观察结果调整：

**如果 BUY 信号太少：**
```json
{
  "min_confidence": 0.70  // 从 0.75 降到 0.70
}
```

**如果交易过于频繁：**
```json
{
  "min_confidence": 0.75,  // 提高阈值
  "check_interval_minutes": 10  // 延长检查间隔
}
```

**如果止损过大：**
```json
{
  "default_stop_loss_percent": 5.0  // 收紧止损（如果系统支持）
}
```

## ⚠️ 风险管理

### 每日检查：

- [ ] 查看今日交易次数（是否达到上限）
- [ ] 查看今日盈亏（是否超过止损限制）
- [ ] 查看持仓数量（是否过度集中）
- [ ] 查看止损止盈执行情况

### 异常情况处理：

**情况1：连续止损**
```
连续止损 ≥ 3 次
→ 立即停止引擎
→ 检查 AI 决策逻辑
→ 调整参数或暂停交易
```

**情况2：单日亏损超限**
```
单日亏损 ≥ max_loss_per_day
→ 系统会自动停止交易
→ 等待次日重新评估
```

**情况3：持仓被套**
```
持仓亏损 ≥ 10%
→ 检查是否应该止损
→ 查看 AI 的卖出建议
→ 必要时手动平仓
```

## 🛠️ 故障排查

### 常见问题：

1. **AI 一直返回 HOLD**
   - 查看 chain_of_thought 了解原因
   - 降低 min_confidence 到 0.70
   - 确认 K线数据充足（≥20根）

2. **提示"已有持仓"**
   - 前往持仓管理 → 删除对应持仓
   - 或点击「清空所有持仓」

3. **真实交易订单失败**
   - 检查 Longbridge 凭据是否过期
   - 确认账户资金充足
   - 查看 logs/backend.log 错误详情
   - 确认在交易时间内

4. **Chain of Thought 为空**
   - 检查 AI 模型版本（需要 deepseek-chat）
   - 查看后端日志确认 API 调用成功
   - 尝试重启引擎
